Running KMM:
------------

To correctly execute the application after the SVN tree download, you should
execute the genfiles.py application. This tool will generate all files related
to the UI and resources and is needed for the execution.

Development Requirements:
-------------------------

I currently develop on windows so no tests on other platforms has been made.

- Python 2.4.3 install
- PySQLite2 module
- Tolltech QT 4.1.4 opensource edition
- PyQt 4.0.1 module - Qt bindings for Python
- Py2exe module to build executables
- Innosetup to build the installer
